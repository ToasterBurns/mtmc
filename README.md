# MobTracker for Minecraft 1.21.10

A Fabric mod that tracks entities with specific nametags in Minecraft.

## Features
- Press `H` to toggle tracking
- Track entities with configurable nametags
- In-game configuration GUI (Press `J`)
- Add nametags by looking at entities (Press `K`)
- Chat commands for control

## Building
1. Open in GitHub Codespaces
2. Run: `./gradlew build`
3. Find JAR in: `build/libs/mobtracker-1.0.0.jar`

## Installation
Place the JAR file in your `.minecraft/mods/` folder.
